## ✨ Features

- **Bilingual (Gujarati + English)** — Full `lang="gu"` support with Noto Sans Gujarati fonts
- **Mobile-first responsive** — Tested from 320px to 1400px
- **WhatsApp CTA integration** — Pre-filled order messages in Gujarati
- **SEO-optimised meta** — Local business schema-ready
- **Performance** — Single-file HTML, zero JS dependencies, CDN fonts only
- **Pure Veg promise strip** — Prominent 100% vegetarian branding
- **Tiffin delivery section** — Daily subscription flow
- **Stats strip** — 500+ daily customers, unlimited servings
- **Competitor-aware** — Built after auditing 6 top Surat/NJ Gujarati thali brands
